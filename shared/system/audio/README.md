# PipeWire

Installs the PipeWire audio stack.

---

## Installation

```bash
./install.sh
````

---

## Includes

* pipewire
* wireplumber
* pipewire-alsa
* pipewire-pulse
* pavucontrol

---

## Configuration

User services are enabled automatically.

---

## Notes

* replaces PulseAudio and ALSA user layers
* log out and back in if audio is not detected
