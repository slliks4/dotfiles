# Fonts

Installs system fonts.

---

## Installation

```bash
./install.sh
````

---

## Includes

* terminus-font
* spleen-font (AUR)

---

## Configuration

Fonts are installed system-wide.

No additional configuration is required.

---

## Notes

* uses `paru` for AUR packages
* installs `paru` if not available
* fonts are minimal and terminal-focused
