# GPU

GPU setup is split by vendor.

Each implementation is independent and should be selected based on hardware.

---

## Vendors

### Intel

Integrated and supported out of the box on most systems.

See:
- `intel/README.md`

---

### AMD

Open-source drivers with kernel support.

See:
- `amd/README.md`

---

### NVIDIA

Proprietary drivers with additional configuration requirements.

See:
- `nvidia/README.md`

---

## Notes

- install only one vendor configuration
- do not mix driver stacks
- refer to vendor modules for installation details
